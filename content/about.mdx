This is a minimal MDX starter template built with Next.js, Velite, and Tailwind CSS. It's designed to be simple, fast, and easy for AI to work with.

## Features

- **MDX Support** - Write content in Markdown with React components
- **Velite** - Fast content processing and type-safe data
- **Tailwind CSS** - Utility-first styling
- **Dark Mode** - Built-in theme switching
- **Syntax Highlighting** - Code blocks with highlight.js

## Getting Started

1. Clone the repository
2. Install dependencies with `bun install`
3. Run `bun dev` to start the development server
4. Add your content in the `content/` directory

## File Structure

```
content/          # MDX content files
components/       # React components
app/              # Next.js app router
lib/              # Utilities
```

## Writing Content

Create `.mdx` files in the `content/` directory with frontmatter:

```mdx
---
title: "My Post"
description: "A description"
date: "2025-01-01"
published: true
---

Your content here...
```

That's it. Keep it simple.
